2. map 结构

   - 由于shell的的数组只支持一维数组**,并且并没有map这种数据结构也。我们使用关联数组来模拟map的效果。首先定义关联数组(可以以字符串为下标): delcare -A xxx
     delcare 的使用：

   - ```
     #!/bin/bash
     declare -A map
     
     map["key1"]="value1"
     map["key2"]="value2"
     map["key3"]="value3"
     
     # 遍历map,此map其实就是数组，其相关操作同上面数组的操作
     for key in ${!map[@]};do
         echo "key: "${key}" value:"${map[${key}]}
     done
     ```
