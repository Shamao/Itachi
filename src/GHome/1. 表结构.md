#### 物品表 goods	
| 字段            | 类型    | 说明        |
| --------------- | ------- | ----------- |
| id              | integer | id          |
| name            | String  | 名称        |
| count           | int     | 数量        |
| unit            | string  | 单位        |
| production_date | date    | 生产日期    |
| shelf_life      | int     | 保质期 月份 |
| userid          | int     | 用户id      |
| source          | int     | 购买来源    |
| img_url         | String  | 图片资源    |
